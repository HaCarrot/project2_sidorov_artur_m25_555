# Управление таблицами
## Создание таблицы

**Команда:** ```create <имя_таблицы> <столбец1> <тип1> [<столбец2> <тип2> ...]```

Создает новую таблицу с указанным именем и структурой. Автоматически добавляет столбец ID:int в качестве первичного ключа.

Допустимые типы данных:
```int``` - целые числа
```str``` - строки
```bool``` - логические значения (true/false)

## Просмотр списка таблиц

**Команда:** ```list```

Выводит список всех таблиц в базе данных с их структурой.

## Удаление таблицы

**Команда:** ```drop <имя_таблицы>```

Удаляет указанную таблицу из базы данных.
