# Управление таблицами
## Создание таблицы

**Команда:** ```create <имя_таблицы> <столбец1> <тип1> [<столбец2> <тип2> ...]```

Создает новую таблицу с указанным именем и структурой. Автоматически добавляет столбец ID:int в качестве первичного ключа.

Допустимые типы данных:
* ```int``` - целые числа
* ```str``` - строки
* ```bool``` - логические значения (true/false)

## Просмотр списка таблиц

**Команда:** ```list```

Выводит список всех таблиц в базе данных с их структурой.

## Удаление таблицы

**Команда:** ```drop <имя_таблицы>```

Удаляет указанную таблицу из базы данных.

# CRUD - операции

## Вставка данных (INSERT)

**Команда:** ```insert into <имя_таблицы> values (<значение1>, <значение2>, ...)```

Добавляет новую запись в таблицу. ID генерируется автоматически.

## Выборка данных (SELECT)

**Команда:** ```select from <имя_таблицы>```

При необходимости выборки с условием используйте команду: ```select from <имя_таблицы> where <столбец> = <значение>```

Выводит записи в таблице.

## Обновление данных (UPDATE)

**Команда:** ```update <имя_таблицы> set <столбец> = <новое_значение> where <условие>```

Обновляет записи, соответствующие условию.

## Удаление данных (DELETE)

**Команда:** ```delete from <имя_таблицы> where <условие>```

Удаляет записи, соответствующие условию.
